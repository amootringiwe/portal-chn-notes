# portal-chn-notes

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以笔记或工具页的形式存在，不针对任何特定网站或域名，仅作为静态资源存放与访问的载体。

## 项目简介

portal-chn-notes 是一个以 HTML 页面为主要输出形式的归档仓库。每个页面均为独立文件，不依赖后端服务或数据库，可直接通过浏览器打开或托管于静态页面服务上使用。

仓库主要服务于个人或小范围共享场景，适合存放临时性、工具性、参考性的网页内容。

## 目录说明

```
.
├── index.html          # 仓库入口页（可选）
├── pages/              # 页面归档目录
│   ├── page-001.html
│   ├── page-002.html
│   └── ...
└── README.md           # 本文件
```

- `index.html`：可选的主页，用于展示页面列表或导航。
- `pages/`：存放所有独立 HTML 页面，按需命名，不强制统一前缀。

## 页面归档说明

- 每个页面均为自包含的 HTML 文件，所有样式、脚本均内嵌或引用自 CDN 资源。
- 页面内容不涉及任何具体商业、推广或敏感信息。
- 页面命名建议简洁明了，避免使用特殊字符。

## 维护说明

- 本仓库由维护者不定期更新，不保证页面内容的持续可用性或准确性。
- 如需新增页面，请将 HTML 文件放入 `pages/` 目录，并可选更新 `index.html` 中的导航链接。
- 不鼓励对本仓库进行大规模自动化抓取或镜像。

## 许可

本仓库内容仅供学习与参考使用。各页面内可能引用的第三方资源（如字体、图标、库文件）遵循其各自的许可协议。
